I'm a Next.js fullstack web developer. This is a sample production grade project by Me.

I used:

✅ Next.js App Router
✅ MongoDB Via Prisma
✅ Stripe Integration
✅ Zustand, Tailwind, NextAuth, React Query
✅ NextAuth v5
✅ Shadcn/ui
✅ React Hook Form
✅ Sanity
✅ Redis
✅ Cloudinary
✅ Typescript

This Project has:

» Project Listing
» Variant and Category Management System
» Stripe Payment Flow
» Cash-On Delivery Flow
» Admin Panel
» Real order model 
» Reporting System
» Shopping & Cart Management System
» Advanced Customer and Admin Dashboard
